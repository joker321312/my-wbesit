# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

**Email:** security@toolhub.dev

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We aim to respond within 48 hours and will coordinate disclosure with you.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |

## Security Notes

- All payments are crypto-only (no traditional payment processors)
- Admin panel requires password authentication
- Download links are generated with unique tokens
- No sensitive user data stored beyond order records
